## Краткое содержание

Учебное задание: выполнить один и тот же запрос с temperature = 0, 0.7 и 1.2, сравнить ответы по точности, креативности и разнообразию, сформулировать выводы о применимости каждой настройки.

## Требования задания

- Запрос с temperature = 0 (детерминированный)
- Запрос с temperature = 0.7 (баланс)
- Запрос с temperature = 1.2 (творческий)
- Сравнение по точности, креативности, разнообразию
- Вывод: для каких задач подходит каждый режим

## Статус в NeuroChat

✅ **Реализовано** — задание учебное (эксперимент), не требует новой фичи

Температура полностью настраивается через `TemperatureSlider` в `SettingsPanel`:
- Диапазон: 0.0–2.0, шаг 0.1
- Все три значения (0, 0.7, 1.2) доступны
- Сохраняется в DataStore, передаётся в `ApiSettings.temperature` → `ChatRequest.temperature`

Для выполнения задания достаточно выставить нужное значение и отправить один и тот же текст.
